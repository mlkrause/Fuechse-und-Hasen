Projekt: Fuechse-und-Hasen-V2
Autoren: David J. Barnes und Michael Kölling

Dieses Projekt ist Teil des Zusatzmaterials zum Buch

   Java lernen mit BlueJ - eine Einführung in die 
   objektorientierte Programmierung, 3. Auflage
   David J. Barnes und Michael Kölling
   Pearson Education Deutschland, 2006

Eine Jäger-Beute-Simulation mit Füchsen und Hasen, die sich
in einem rechteckigen Feld bewegen.

Dies ist die zweite Version der Simulation. In dieser Version
wird Vererbung eingesetzt, um die Struktur des Quelltextes
zu verbessern.

Startpunkt:
    Erzeugen Sie ein Objekt der Klasse Simulator.
    Rufen Sie dann eine dieser Methoden auf:
        + simuliereEinenSchritt - für einen einzelnen Schritt.
        + simuliere - mit Angabe der zu simulierenden Anzahl an Wochen.
        + starteLangeSimulation - für eine Simulation von 500 Wochen.
